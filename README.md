# Simple Image Editor

A simple web-based image editor built using HTML, CSS, and JavaScript.

## Features

* Adjust brightness and saturation
* Apply grayscale and inversion filters
* Rotate and flip images
* Simple and user-friendly interface
* Works without login

## Technologies Used

* HTML
* CSS
* JavaScript
* PHP (for visit counter)

## How to Use

1. Open the project in your browser
2. Upload an image
3. Apply filters and adjustments
4. Download the edited image

## Project Purpose

This project was created to provide a lightweight and easy-to-use image editing tool without requiring any login or installation.
Note: PHP-based visit counter works only in local/server environment and not on GitHub Pages. please download and run index.php manually offline.thanking you!
